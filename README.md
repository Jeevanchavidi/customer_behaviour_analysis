# 📌 Project Overview

Customer behavior analysis helps businesses understand purchasing patterns, improve customer retention, and make informed business decisions. This project analyzes an e-commerce customer dataset containing **3,900 customer records** to identify customer segments, revenue trends, product performance, and purchasing behavior.

The project follows a complete analytics workflow—from data cleaning and preprocessing in Python to SQL-based business analysis and interactive dashboard creation in Power BI.

---

# 🎯 Objectives

- Clean and preprocess raw customer data
- Handle missing values and improve data quality
- Perform Exploratory Data Analysis (EDA)
- Build an automated Python to MySQL pipeline
- Solve real-world business problems using SQL
- Build an interactive Power BI dashboard
- Generate actionable business insights and recommendations

- Removed unnecessary columns
- Handled missing values
- Standardized column names
- Removed duplicate records
- Created Age Group categories
- Converted Purchase Frequency into numerical values
- Checked data consistency
- Prepared the dataset for SQL analysis and Power BI

---

# 📈 SQL Business Questions

This project answers the following business questions:

1. Revenue generated by Gender
2. High-value customers using discounts
3. Top 5 highest-rated products
4. Standard vs Express Shipping comparison
5. Subscriber vs Non-Subscriber spending
6. Products with the highest discount rate
7. Customer Segmentation using CTE
8. Top Products using Window Functions
9. Repeat Buyer Analysis
10. Revenue by Age Group

# 📌 Key Insights

## Customer Insights

- Clothing generated the highest revenue.
- Young Adults contributed the highest sales.
- Subscribers spent more on average than non-subscribers.
- Loyal customers were more likely to purchase subscriptions.

## Product Insights

- Clothing was the best-performing category.
- Blouse, Jeans, and Handbag received the highest customer ratings.
- Highly rated products showed stronger customer demand.

## Business Insights

- High-value customers frequently used discounts.
- Customer segmentation supports targeted marketing campaigns.
- Interactive dashboards improve decision-making through real-time business insights.

---

# 📌 Business Recommendations

- Increase marketing investment in high-performing product categories.
- Launch loyalty programs to improve repeat purchases.
- Offer exclusive subscription benefits to increase customer retention.
- Promote highly rated products using targeted campaigns.
- Develop personalized offers for young adult customers.
